# 🎯 TASKFORGE - Super Simple Setup (For Friends)

## What is this?

TASKFORGE is a **team collaboration app** that helps you:
- ✅ Manage tasks together
- ✅ Use AI to write task descriptions automatically
- ✅ Have video meetings with your team
- ✅ Earn points and badges for completing tasks

---

## 🎬 What You'll Need (All FREE!)

### 1. **Node.js** (Required)
- **What:** Software that runs JavaScript code
- **Download:** https://nodejs.org
- **Time:** 2 minutes
- **Size:** ~50 MB

### 2. **MongoDB Account** (Required - FREE Forever)
- **What:** Online database to store your data
- **Sign up:** https://www.mongodb.com/cloud/atlas
- **Time:** 5 minutes
- **Cost:** $0 (FREE tier)

### 3. **Groq API Key** (Required - FREE)
- **What:** Powers the AI features
- **Get key:** https://console.groq.com
- **Time:** 2 minutes
- **Cost:** $0 (7,200 free requests/day)

### 4. **Gmail Account** (Required - You probably have one!)
- **What:** Sends verification emails
- **Time:** 2 minutes to setup app password
- **Cost:** $0

## ⚡ Installation (10 Minutes)

### Step 1: Install Node.js
1. Download from https://nodejs.org (get the LTS version)
2. Run installer
3. Keep clicking "Next"
4. Done!

**Check it worked:**
- Open Terminal/Command Prompt
- Type: `node --version`
- Should show: `v18.x.x` or similar ✅

---

### Step 2: Get Your Keys

#### 🗄️ MongoDB
1. Sign up at https://www.mongodb.com/cloud/atlas
2. Create FREE cluster
3. Create database user (username: `taskforge`, auto-generate password)
4. Allow access from anywhere
5. Get connection string (looks like `mongodb+srv://...`)
6. **IMPORTANT:** Replace `<password>` with your actual password!

#### 🤖 Groq
1. Sign up at https://console.groq.com
2. Create API key
3. Copy it (starts with `gsk_`)

#### 📧 Gmail
1. Go to https://myaccount.google.com/apppasswords
2. Create app password for "Mail"
3. Copy the 16-digit code

---

### Step 3: Setup Backend

**Open Terminal/Command Prompt and type:**

```bash
# Go to the folder you extracted
cd Desktop/taskforge-main

# Go to backend folder
cd backend

# Install packages (wait 1-2 minutes)
npm install
```

---

### Step 4: Configure

1. In `backend` folder, copy `.env.example` to `.env`
2. Open `.env` file
3. Paste your keys:

```env
MONGODB_URI=mongodb+srv://taskforge:YOUR_PASSWORD@...
EMAIL_USER=your.email@gmail.com
EMAIL_APP_PASSWORD=abcdefghijklmnop
GROQ_API_KEY=gsk_your_groq_key_here
```

4. Save!

---

### Step 5: Create Admin

```bash
node quickAdminSetup.js
```

Should say: `✅ Admin Created Successfully!`

---

### Step 6: Setup Frontend

```bash
# Go to frontend folder
cd ../frontend

# Install packages (wait 2-3 minutes)
npm install
```

---

### Step 7: Start!

**Open 2 Terminal windows:**

**Terminal 1:**
```bash
cd backend
npm run dev
```

**Terminal 2:**
```bash
cd frontend
npm start
```

Browser opens automatically! 🎉

## 🐛 Problems?

### "npm not found"
→ Install Node.js first!

### "Cannot connect to MongoDB"
→ Check your `.env` file, make sure password is correct

### "Email not sending"
→ Check Gmail app password (no spaces!)

### "Groq AI not working"
→ Check API key starts with `gsk_`
